# rust-plugin-extism

A sample project using [Extism](https://extism.org/) for Rust

### install Extism CLI

```shell
curl -s https://get.extism.org/cli | sh
```

### add rust Wasm target

```shell
rustup target add wasm32-unknown-unknown
```

### build sample

```shell
cargo build --target wasm32-unknown-unknown
```
